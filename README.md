# 📣 TyIllégal 
*- Version 1.0.0*
# ⚜️ Fonctionnalités 
*- Permet de mettre un point dans la map permettant d'acheter des graine de cannabis et des feuilles de cannabis*
*- L'emplacement du point est configurable dans TyIllégal.json*
*- Le prix pour acheter les feuilles est configurable tout comme le prix pour acheter des graines de cannabis*
# 📦 Installation 
*- Téléchargez TyIllégal.dll puis glissez le dans le dossier plugin de votre serveur , lancez votre serveur une première fois puis configurez le plugin dans le fichier Tyillégal.json puis relancez votre serveur.*
